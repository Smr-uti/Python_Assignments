I1. What is a tuple in Python? Define it in one sentence.
:-- A Tuple is ordered, immutable, collection of elements created using round brackets().

I2. Which brackets are used to create a tuple?
:-- Round brackets () are used to create tuple.

I3. What is the difference between a homogeneous tuple and a heterogeneous tuple? Give one example of each.
:-- Homogeneous Tuple-
Homogeneous tuple is a collection of elements that are all of the same data type.

Heterogeneous Tuple- 
Heterogeneous tuple is a collection of finite, ordered sequence of elements where each element can be of a different data type.

I4. What does "ordered" mean for a tuple?
:-- Ordered means that the elements in a tuple have a fixed position (index), and that order is always maintained.

● Each element can be accessed by its index, starting from 0.
● The order in which you insert elements is the order they stay — it never changes.

7. What are the three operations you CANNOT perform on a tuple because it is immutable?

    ● Remove an element — remove(), pop(), or del t[i] won't work
    ● Change/replace an element — you cannot reassign by index (t[0] = 99)

while loop — manual and controlled:
