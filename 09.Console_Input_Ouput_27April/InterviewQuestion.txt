Q1. What is Console I/O in Python? What is the difference between input and output?
:-- Console Input/Output allows programs to user interact with users.
Console I/O (Input/Output) refers to the process of exchanging data between a computer program and the user via the 
command-line interface or console.
i] input()- input() functions takes input from the user and returns it as a string.
We can also pass a string as an argument to the input() function which will be displayed as a prompt to the user.
ii] output()- output() function display output to the console/screen.

Q6. What does the input() function do? What data type does it always return?
:-- input() function takes input from the user & returns it as a string.
input() function is used to take user input. When this function is called, the program pauses its execution and waits for the user to
type something into the console and press Enter.
No matter what the user types—whether it is a word, a sentence, a whole number, or a decimal—the input() function always returns that 
value as a string (str) data type.

Q11. What is the difference between input() with a prompt string and input() without one?
:-- i] input() with a prompt string
When you pass a string inside the parentheses, Python displays that text on the screen before pausing. This tells the user exactly what information the program is waiting for.

Ex. # With a prompt string
      name = input("Please enter your name: ") #Display custom message
      print("Hello, " + name)
Output: Hello, Smruti

Ex. # Without a prompt string
      name = input()
      print("Hello, " + name) #Displays the black screen.
Output: Hello, Smruti
