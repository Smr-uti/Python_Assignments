Q1. What is decision control in Python? Why do we need it?
:-- Decision Control refers to the ability of a program to make "decisions" based on certain conditions. It allows your program 
to execute different blocks of code depending on whether a specific statement is true or false.
Without decision control, a program would be a "straight line"—it would execute every line of code from top to bottom, in the 
same order, every single time.

Q3. What is the difference between if and if-else statement?
:-- i] if statement:
To run code only on a specific condition.
If the condition is false it does nothing; skips the block.
Provides one path for execution.

ii] if-else statement:
To choose between two different condition.
Executes the else block.
Provides two path for execution.

Q4. What is the difference between if-else and if-elif-else?
:-- i] if-else statement:
Exactly two conditions.
Only one condition checked.

ii] if-elif statement:
Three or more conditions.
Multiple sequential conditions checked.

Q5. How many elif blocks can you write in a single if-elif-else statement?
:-- You can write as many elif blocks as you need in a single if-elif-else structure. There is no technical limit imposed by 
Python. Whether you need to check 3 conditions or 300, Python will handle them sequentially.

Q6. What is indentation in Python? What happens if you do not indent the code inside an if block?
:-- Indentation tells the Python interpreter which lines of code belong together. When you write an if statement, the block 
of code that is "inside" that condition must be indented to show that it is a dependency of that condition.
If you fail to indent the code inside an if block, Python will raise an IndentationError and your program will refuse to run.

Q7. What is the difference between the and operator and the or operator?
:-- The and and or operators are logical operators used to combine multiple conditions.
i]and operator: 
The and operator returns True only if both conditions are True. If even one condition is False, the entire 
result becomes False.

ii]or operator:
The or operator returns True if at least one of the conditions is True. It only returns False if both conditions are False.

:-- Yes, you can absolutely write an if statement without an else block.
No, else is not mandatory. An if statement can exist entirely on its own.
When to use only if: You use it when you only want to perform an action if a specific condition.

Q13. What happens if none of the elif conditions are True and there is no else block?
:-- If none of the if or elif conditions evaluate to True and there is no else block present, nothing happens.
The entire conditional structure is simply skipped, and the program execution proceeds directly to the very next line of code 
outside of that structure.
