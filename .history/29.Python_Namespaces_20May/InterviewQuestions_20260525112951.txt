Q1.  Write out the LEGB rule in full. What does each letter stand for? In what order does Python search each level?
:-- L (Local): Inside the current function.
E (Enclosing): In any outer functions (nested scopes).
G (Global): At the top level of the script/module.
B (Built-in): In Python’s pre-defined library of functions and keywords.

Search order for each level:
Local: Python first looks for the name in the current local scope
Enclosing: If not found, it moves outward to the scope of any enclosing functions
Global: If still not found, it checks the global scope
Built-in: Finally, it searches the built-in scope, which contains all of Python's reserved keywords and standard functions.


Q2.  What is the enclosing (E) namespace? Give a clear example showing a variable in the enclosing scope.
:-- Enclosing(E):
The Enclosing (or nonlocal) scope is the scope of any outer function that contains a nested function. It comes into play when you 
have a function defined inside another function; the inner function has access to the variables defined in the outer function's 
local scope.

Ex>: def outer_func():
     message = "I am in the enclosing scope!"  # Enclosing scope variable
    
          def inner_func():
          print(message)  # Accesses the variable from the enclosing scope
    
          inner_func()
    outer_func()
Output: I am in the enclosing scope!
