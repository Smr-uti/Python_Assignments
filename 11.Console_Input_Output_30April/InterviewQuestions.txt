Q6. Does the order of variables inside format() matter? What happens if you change the order?
:-- Yes, the order of variables inside the format() method absolutely matters because, by default, Python maps them to the
placeholders ({}) in the order they appear. If you change the order inside the format() method without changing the placeholders 
in the string, the values will be swapped.
