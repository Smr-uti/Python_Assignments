7. Why is Python called a cross-platform language?
:-- Because Write once, Runs anywhere. 
This means that code written on one operating system (like Windows) can run on another (like macOS or Linux) without needing to change the actual code.

8. What is CPython?
:-- CPython is the default and most widely used implementation of the Python programming language. CPython is the interpreter. It is the actual software program that reads your Python code and executes it.

9. What is the role of the Python Virtual Machine (PVM)?
:-- Python Virtual Machine (PVM) is the engine that actually runs your code. It is the "runtime" part of the Python system that takes the translated instructions and executes them on your computer's hardware.

10. What is the difference between machine code and bytecode?
:-- Machine code:
Machine code is the most basic, fundamental language of a computer. It is the "native tongue" of the computer's CPU (Central Processing Unit).

ByteCode:
Bytecode is an intermediate step. It is a set of instructions that is more advanced than machine code but simpler than the Python code you write.
