Q2. What is the difference between lstrip(), rstrip(), and strip()? Give one example of each.
:-- These three methods are used to remove whitespace (or specific characters) from the ends of a string.
i] lstrip()- It removes leading left whitespaces.
Ex. text1= "   Python Programming   "
    print(text1.lstrip())
output-Python Programming   .

ii] rstrip()- It removes trailing/right side whitespace.
Ex. text2="   Python Programming   "
    print(text2.rstrip())
output:   Python Programming.

iii] strip()- It removes leading and trailing (left & right) whitespaces.
Ex. text3="  Python Programming  "
    print(text3.strip())
Output: Python Programming.
