Q11. What does the `len()` function return for a string?
:-- the len() function returns the total number of characters in a string. It treats the string as a sequence and counts 
everything inside the quotes, including:
Letters (a-z, A-Z)
Digits (0-9)
Symbols/Punctuation (@, #, !, etc.)
Spaces (each empty space counts as one character)
Escaped characters (like \n for a new line, which counts as one character)

Q12. How do `min()` and `max()` work on strings?
:-- i]'min()':
It returns smallest character from the string.
ii] 'max()':
It returns the item with highest value.
With Numbers: It finds largest numbers.
With Strings: It finds the character that comes latest in alphabetical order (based on ASCII values).
